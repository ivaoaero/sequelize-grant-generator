# Sequelize Grant Generator

Typescript plugin that extracts Sequelize models usage in source code and generates SQL grants accordingly

## Context

This tool was built by and for the [IVAO](https://www.ivao.aero/) Web Development team to automatically grant its API the rights accesses to the databases and tables based on the models used in the code.
